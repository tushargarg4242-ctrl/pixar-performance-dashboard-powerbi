# 🎬 Pixar Performance Dashboard (Power BI Project)

## 📌 Overview

This project presents an interactive **Power BI dashboard** analyzing the performance of Pixar films across financial, audience, and critical success metrics.

The goal of this project is to identify:

* What drives **profitability (ROI)**
* How **audience ratings have changed over time**
* Which **genres and films perform best**

---

## 📊 Key Metrics

* 💰 **Total Revenue:** $17.04B
* 📈 **Total Profit:** $12.54B
* 🎥 **Total Films:** 28
* 📊 **Average ROI:** 317.93%

---

## 🧩 Dashboard Pages

### 1️⃣ Financial Performance

* Revenue vs Budget trends over time
* Profit fluctuations and recovery patterns
* Top-performing films based on ROI

👉 **Insight:**
Older films like *Toy Story* and *Finding Nemo* deliver significantly higher ROI (>500%) compared to recent releases.

---

### 2️⃣ Genre & Awards Analysis

* ROI comparison across genres
* Revenue contribution by genre and films
* Academy Awards (Wins vs Nominations) trend

👉 **Insight:**
Genres like **Buddy Comedy** and **Adventure** generate the highest returns, while awards peaked between **2005–2010**.

---

### 3️⃣ Audience Analysis

* IMDb rating trends over time
* Runtime vs rating relationship
* Film-level performance summary

👉 **Insight:**
Audience ratings declined from **~8.3 (early 2000s)** to **~7.0–7.6 (recent years)**, showing reduced audience satisfaction.

---

## 🔍 Key Findings

* 🚀 Pixar’s **golden era (2000–2010)** delivered:

  * Highest ROI
  * Strongest audience ratings
  * Maximum awards

* 📉 Post-2015 trends show:

  * Declining ROI efficiency
  * Lower audience ratings
  * Fewer award wins

* ⚠️ High revenue does not always mean high profitability —
  **ROI is the true performance indicator**
